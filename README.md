# Grid Telemetry

Production-grade F1 monitoring system inspired by industrial pitwall telemetry.

## Overview

Grid Telemetry is a telemetry platform for ingesting, processing, and presenting race data in near real time.

## Getting Started

### Stack

- FastAPI
- Python 3.14
- uv
- OpenF1 API
- Redis
- PostgreSQL
- Vue 3
- Nuxt 3
- Tailwind
- shadcn-vue
